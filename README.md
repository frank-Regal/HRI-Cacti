# HRI-Cacti
Human Robot Interaction (HRI) Command and Control Teammate Interface (CACTI) project hub.

## Getting started
1. Clone this repo
```shell
$ git clone git@github.com:frank-Regal/HRI-Cacti.git
```
2. Setup environmental variables
```shell
$ cd HRI-Cacti
$ ./setup.sh
```
3. Setup docker
```shell
$ doc-build
$ doc-start
$ doc-shell
```
